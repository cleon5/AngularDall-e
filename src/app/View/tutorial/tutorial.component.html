Comienza a usar dall-e

1.- RapidAPI

Inicia seccion o registrate en RapidAPI, para conseguir la apiKey https://rapidapi.com/hub

2.- Ingresar a OpenAI

Deberas buscar openAi en la seccion de busqueda, y entra al primer resultado

3.- Activar api

Deberas seleccionar la pricing y subscribirte al plan basico, ya que este es el gratuito

4.- Consigue la apikey

Regresa a la seccion de Endpoints, y deslizate hacia abajo para encontrar el Api-key y el Host

5.- Pasa a crear

Teniendo el apiKey y el Host ya puedes comenzar a crear tus imagenes Empieza a crear
